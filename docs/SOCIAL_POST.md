## Short Launch Post

AI agents often stop too early.

You ask an agent to work until 06:00. It finishes the first task in 20 minutes, then says "done."

Owlyn MCP gives agents a persistent work-session clock, checkpoints, deadlines, and structured continuation decisions so they know when to keep going safely.

v0.1.0-alpha.1 is now public:
https://github.com/btitkin/OwlynMCP

Local-first. SQLite. stdio MCP. No cloud. No telemetry.

## Shorter Version

Owlyn MCP is a local-first work-session supervisor for AI agents.

It helps agents remember the session deadline, checkpoint progress, and decide whether to continue instead of stopping after the first completed task.

Public alpha:
https://github.com/btitkin/OwlynMCP
